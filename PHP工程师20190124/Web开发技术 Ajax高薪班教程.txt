AJAX = Asynchronous JavaScript and XML（异步的 JavaScript 和 XML）。

AJAX 不是新的编程语言，而是一种用于创建更好更快以及交互性更强的Web应用程序的技术。AJAX通过在后台与服务器进行少量数据交换，AJAX 可以使网页实现异步更新。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新。是与服务器交换数据并更新部分网页的艺术。

在学习本套课程之前，你需要掌握：

HTML / XHTML
CSS
JavaScript / DOM
